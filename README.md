## ✨ Features

- **Add Expenses**: Quickly log your daily expenses with descriptions and amounts
- **Categorize Spending**: Organize expenses by category for better tracking
- **View Expense List**: See all your expenses in a clean, organized format
- **Delete Expenses**: Remove entries with a single click
- **Calculate Totals**: Automatically compute total spending
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Local Storage**: Your expenses are saved automatically in your browser
- **Modern UI**: Clean, professional interface with smooth interactions
- **Real-time Updates**: Instant feedback on all operations

## 📦 Installation

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server or backend required

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/Nancyriya27/Expense-Tracker.git
   cd Expense-Tracker
   ```

2. **Open the application**
   - Option 1: Double-click `index.html` to open in your default browser
   - Option 2: Use a local server (recommended)
     ```bash
     # Using Python
     python -m http.server 8000
     # or using Node.js
     npx http-server
     ```
   - Open `http://localhost:8000` in your browser

## 💡 Usage

### Adding an Expense

1. Enter the expense description in the text field
2. Enter the amount spent
3. Select a category (optional)
4. Click the "Add Expense" button
5. Your expense will appear in the list below

### Managing Expenses

- **View Total**: See your total spending displayed at the top
- **Delete Expense**: Click the delete button on any expense to remove it
- **Clear All**: Use the clear button to remove all expenses

### Data Persistence

Your expenses are automatically saved to your browser's local storage, so they'll be available even after you close and reopen the application.
